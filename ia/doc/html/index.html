Introduction

This documentation will help non-developpers or developpers to implements new Artificial Intelligence on the Bomberman.

Installation

Tools required:

GDL available here with the folowing command : "scp -r ssh.epitech.net:/u/all/gamelab/public/libgdl_gl-2012.3.tgz .".
Untar "tar -xvf libgdl_gl-2012.3.tgz" and copy the library in /usr/lib and the headers files in /usr/includes.
Type "make" in the console to compile the program.

Running the program

Once the compilation is done, simply type ./Bomberman in the console to lauch the game !
